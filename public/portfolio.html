Workday Planner

This application is designed as a workday planner that lets users assign and save periods of time in their workday to better prioritize and plan out their day accordingly.

WeatherJams

This application is a simple user interface that lets users not only view the weather in their current location as well as other cities but it gives users the option to play music that corresponds with the current weather conditions.

Burger App

This project was designed using a MySQL database. It is made to let users enter their favorite burger into the field which transports that data to store in the database. This project taught me how SQL databases communicate with front-end code.

Happy Paws

This project was a collaboration effort, we created a dog day care website which allows users to create an account and book reservations for their dog's stay.
